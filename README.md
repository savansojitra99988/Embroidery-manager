This project is a Progressive Web App (PWA) built with vanilla JavaScript, HTML, and CSS. It helps embroidery businesses calculate worker bonuses based on stitch counts and manage attendance records.

Key Features:

Authentication: Secure login/signup with Firebase Realtime Database

Cloud Sync: Data automatically stored and synced across devices using Firebase

Cross-Device Access: Works seamlessly on mobile, tablet, and desktop

Offline-First: Service Worker caching ensures the app runs without internet; updates sync when reconnected

Bonus Calculator: Calculates bonus amounts based on number of stitches

Team & Worker Management: Add teams, workers, and track attendance

Backup & Restore: Export/import JSON backups for manual sync if needed

Simple UI: Lightweight, responsive design with real-time updates
